Quick Wiki Template Editor tool

- Its a Simple tool which let you edit various Level3 WiKi templates of any size and with any number of parameters quickly and easily. Useful for IP install and disconnect teams.

- Frequently used NID templates (ESP and PR NID templates for Blue orders) are already fed to the tool. Descriptions of parameters present in these templates is also available.
  If the templates changes or if you want to use other templates please select "Other Template" radio button.

- The tool will automatically fetch various parameters ($----$), will ask for their values and will provide edited template ready to be used.

- It reduces time and overhead of manually searching and replacing parameters, eliminates typo errors, and guarantees that all parameters are replaced without fail.

- Using this tool you can edit existing wiki templates for installs and disconnects. Also you can use it to edit your own templates you might use.
